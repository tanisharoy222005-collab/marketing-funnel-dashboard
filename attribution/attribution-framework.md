# Attribution Framework

## Overview

Attribution analysis determines how marketing channels contribute to lead generation and revenue outcomes.

The purpose of this framework is to provide a consistent methodology for evaluating marketing performance across acquisition channels.

---

# Business Objective

Understand:

* Which channels generate leads
* Which channels generate revenue
* Which campaigns deserve additional investment
* Which touchpoints influence conversion

---

# Attribution Models

## First Touch Attribution

Definition

Assigns 100% credit to the first marketing interaction.

Example

LinkedIn Ad → Organic Visit → Lead

Credit:

LinkedIn Ad = 100%

Use Case

* Awareness measurement
* Top-of-funnel reporting

---

## Last Touch Attribution

Definition

Assigns 100% credit to the final interaction before conversion.

Example

LinkedIn Ad → Organic Visit → Paid Search → Lead

Credit:

Paid Search = 100%

Use Case

* Lead generation reporting
* Performance marketing optimization

---

## Linear Attribution

Definition

Distributes credit equally across all touchpoints.

Example

LinkedIn → Organic → Paid Search

Credit:

33.3% each

Use Case

* Multi-channel reporting

---

## Position-Based Attribution

Definition

Higher credit assigned to first and last interactions.

Example

40% First Touch
40% Last Touch
20% Shared

Use Case

* Full-funnel marketing analysis

---

# Current Model

This project uses:

## Last Touch Attribution

Reason:

* Simplified portfolio implementation
* Commonly used for lead generation reporting
* Easy to explain to stakeholders
