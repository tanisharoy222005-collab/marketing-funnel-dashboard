# Marketing Funnel Dashboard

## Overview

This project demonstrates the design and implementation of a marketing funnel dashboard using GA4, Google Ads, and BigQuery data.

The objective was to provide stakeholders with a single view of acquisition performance, funnel progression, landing page effectiveness, and paid media efficiency.

---

## Business Objectives

* Measure marketing performance across channels
* Identify funnel drop-off points
* Improve lead generation efficiency
* Evaluate paid campaign effectiveness
* Support budget allocation decisions

---

## Funnel Stages

1. Website Session
2. Engaged Session
3. CTA Click
4. Form Start
5. Lead Generation

---

## Key Metrics

* Sessions
* Users
* Engagement Rate
* Conversion Rate
* Leads
* Cost Per Lead
* ROAS

---

## Data Sources

* Google Analytics 4
* Google Ads
* BigQuery Export

---

## Deliverables

* Funnel Dashboard
* SQL Analysis Queries
* KPI Definitions
* Executive Reporting
* Data Blending Documentation

---

## Outcome

The dashboard improved visibility into conversion performance and enabled identification of key funnel bottlenecks affecting lead generation.
